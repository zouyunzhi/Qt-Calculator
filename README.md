# Qt-Calculator
大家好！我是一个11岁的中国男孩。这是我的第一个qt项目————计算器！
经过了大量的修改和优化，它终于完成了！这个计算器不仅功能完整，还使用了强大的exprtk库来处理复杂的数学表达式，而且完全跨平台，可以在Windows、macOS和Linux上运行，甚至有国际化实现。如果大家发现我的计算器有哪些瑕疵，可以指出来，我很乐意接受大家的指点！
注意事项：
1.此项目使用qt6.10.2制作，如果qt版本不足，请到以下网址下载：https://download.qt.io/archive/online_installers/。这个网址存放了许多的qt下载器，而且支持高版本qt，包括qt6.10.2。
