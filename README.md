# Qt-Calculator
大家好！我是一个11岁的中国男孩。我叫邹允之，这是我的第一个qt项目————计算器！

经过了大量的修改和优化，它终于完成了！这个计算器不仅功能完整，还使用了强大的exprtk库来处理复杂的数学表达式，而且完全跨平台，可以在Windows、macOS和Linux上运行。如果大家发现我的计算器有哪些瑕疵，可以指出来，我很乐意接受大家的指点！

注意事项：

1.此项目使用qt6.10.2制作，如果qt版本不足，请到以下网址下载：https://download.qt.io/archive/online_installers/。这个网址存放了许多的qt下载器，而且支持高版本qt，包括qt6.10.2。

2.此项目使用了QSettings，会修改电脑的一些地方，建议不要删除项目。

这个项目没有太多复杂的配置，只要你打开qt creator，打开此项目，接着构建此项目，就能运行啦！

项目特性：

1.此项目没有使用ui，所以代码十分简洁，没有一大堆的槽函数。
2.此项目支持一些快捷键，Enter键直接计算，Esc键清空输入框。
3.此项目支持国际化，默认语言为英语，顶端的菜单栏可以通过“Language”选项来切换语言。
