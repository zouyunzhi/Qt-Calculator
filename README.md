Qt-Calculator

Hello everyone! I'm an 11-year-old Chinese boy. My name is Zou Yunzhi, and this is my first Qt project—Calculator!

After numerous modifications and optimizations, it's finally completed! This calculator is not only feature-complete but also uses the powerful exprtk library to handle complex mathematical expressions. It's fully cross-platform and can run on Windows, macOS, and Linux.

If you find any issues with my calculator, please point them out. I'm more than happy to accept your guidance and suggestions!

Important Notes:

1. Qt Version Requirement: This project is built with Qt 6.10.2. If you don't have this version, please download the Qt online installer from the following URL:

   https://download.qt.io/archive/online_installers/
   
   
   This URL hosts many historical versions of Qt online installers, including versions that support Qt 6.10.2.

2. Settings Storage: This project uses QSettings to save user preferences, which creates configuration files in the system. On Windows, it modifies the registry, so it's recommended not to delete related configurations arbitrarily.

Quick Start:

This project doesn't require complex configuration. You just need to:

1. Install Qt 6.10.2 or higher
2. Open Qt Creator
3. Open this project
4. Click the "Build" button
5. Click the "Run" button

Project Features:

1. Clean Code Structure: This project doesn't use Qt Designer .ui files. All UI elements are created in code, making the codebase very clean without numerous slot functions.

2. Keyboard Shortcut Support:
   • Enter key: Calculate result directly

   • Esc key: Clear the input field

   • Number and operator keys: Direct input

3. Internationalization Support: The default language is English. You can switch between multiple languages using the "Language" option in the top menu bar.

4. Powerful Calculation Engine: Uses the exprtk library to support complex mathematical expression calculations.

5. Cross-platform Compatibility: Built on the Qt framework, it can run on Windows, macOS, and Linux.
